# 🚀 لعبة المغامر الصغير

تطبيق ويب تقدمي (PWA) مصمم خصيصاً للأطفال، يتيح لهم التفاعل مع الكاميرا والتقاط صور مرحة بأسلوب تعليمي وممتع.

## ✨ المميزات
* **تفاعلية:** استخدام الكاميرا بشكل حي ومباشر.
* **تثبيت APK:** يمكن تحويله إلى تطبيق أندرويد وتثبيته على الهاتف.
* **العمل بدون إنترنت:** يدعم ميزة Service Worker للعمل في أي وقت.

## 🛠️ التقنيات المستخدمة
* HTML5 / CSS3 / JavaScript
* ملف التعريف: `manifest.json`
* محرك الأوفلاين: `pwabuilder-sw.js`

## 📲 كيفية التثبيت
1. انسخ رابط الموقع من GitHub Pages.
2. توجه إلى موقع [PWABuilder](https://www.pwabuilder.com).
3. حمل ملف الـ APK وقم بتثبيته على هاتفك.
